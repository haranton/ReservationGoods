# AGENTS.md

При написании и изменении кода обязательно соблюдай стандарты разработки, соглашения и правила оформления, которые описаны в Notion команды.

Если реализация, структура кода, наименование, форматирование, тестирование или архитектурные решения неочевидны, ориентируйся в первую очередь на стандарты из Notion.

Если локальный код противоречит стандартам из Notion, следуй стандартам из Notion, если пользователь явно не указал иное.

Если в модуле для какой-либо области нет кода, не создавай пустую область и не добавляй ее в файл только ради шаблона.

После внесения изменений в код или служебные скрипты всегда выполняй коммит, если пользователь явно не указал иное.
